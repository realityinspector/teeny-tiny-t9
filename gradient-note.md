The gradient of loss w.r.t. layer l's input is ∂L/∂a_l = W_l^T · diag(f') · ∂L/∂a_{l+1}. The singular values of W_l directly control which components of the error signal get amplified vs. suppressed as they backpropagate. If you shape those singular values to be large along directions that correspond to the task's discriminative structure, early gradients are bigger in the directions that matter.
